# AGENTS.md

## 專案目標

WordBridge 是一個給初學者使用的英日語言學習網站。
修改時以簡單、穩定、容易閱讀與擴充為優先。

## 技術限制

- 目前使用原生 HTML、CSS、JavaScript。
- 沒有必要時不要加入框架或第三方套件。
- 學習進度使用瀏覽器 `localStorage` 儲存。
- 不要加入需要 API Key 或後端服務的功能。

## 修改原則

1. 修改前先理解現有結構，不要大幅重寫。
2. 每次只做需求所需的最小修改。
3. 課程資料集中放在 `app.js` 上方的資料陣列。
4. 重複邏輯整理為有意義的函式。
5. 不留下測試碼、無用註解或 `console.log`。
6. 新功能需要同時考慮手機與電腦版。

## 完成後檢查

- 確認 `index.html` 可以正常開啟。
- 執行 `node --check app.js` 檢查 JavaScript 語法。
- 檢查主要導覽、完成課程、測驗與進度儲存。
- 檢查手機寬度下沒有橫向捲動或操作遮擋。
