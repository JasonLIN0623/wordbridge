# WordBridge

WordBridge 是一個不需要安裝套件的英日語言學習網站 MVP。

## 第一版功能

- 英文單字的字首、字根、字尾拆解
- 日文基礎文法、情境常用句與例句
- 英文與日文課程可循環學習，不限制每日練習量
- 可標記不熟內容、立即重複或進入不熟清單集中複習
- 五題英日混合測驗與答題解釋
- 課程完成度、測驗最佳成績與連續學習天數
- 使用瀏覽器 `localStorage` 儲存裝置內進度
- 手機與電腦響應式介面

## 開啟方式

最簡單的方式是直接開啟 `index.html`。

若要使用本機伺服器，可在這個資料夾執行：

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

再開啟：

```text
http://127.0.0.1:8000
```

## 專案結構

```text
.
├── index.html    # 網站結構
├── styles.css    # 版面與響應式樣式
├── app.js        # 課程資料、互動與進度儲存
├── favicon.svg   # 網站圖示
├── AGENTS.md     # 專案開發規則
└── README.md     # 使用說明
```

## 資料說明

所有進度只會儲存在使用者目前瀏覽器的 `localStorage`，不會上傳到伺服器。清除瀏覽器網站資料後，進度也會消失。
