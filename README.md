# WordBridge

WordBridge 是一個不需要安裝套件的英日語言學習網站 MVP。

## 第一版功能

- 60 個英文單字的字首、字根、字尾拆解
- 30 堂日文文法與情境課程，共 60 句原創例句
- 英文與日文課程可循環學習，不限制每日練習量
- 可標記不熟內容、立即重複或進入不熟清單集中複習
- 五題英日混合測驗與答題解釋
- 課程完成度、測驗最佳成績與連續學習天數
- 使用瀏覽器 `localStorage` 儲存裝置內進度
- 手機與電腦響應式介面

## 開啟方式

最簡單的方式是直接開啟 `index.html`。

若要使用本機伺服器，可在這個資料夾執行：

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

再開啟：

```text
http://127.0.0.1:8000
```

## 專案結構

```text
.
├── index.html    # 網站結構
├── styles.css    # 版面與響應式樣式
├── data/

├── favicon.svg   # 網站圖示
├── AGENTS.md     # 專案開發規則

```

## 資料說明

英文課程依使用者提供的 IELTS 4.0–8.5 教材資料夾作為程度與主題分類參考；為避免複製受著作權保護的教材內容，中文解釋與英文例句均重新撰寫。連結內沒有日文教材，因此日文句型與例句為本專案原創內容。

所有進度只會儲存在使用者目前瀏覽器的 `localStorage`，不會上傳到伺服器。清除瀏覽器網站資料後，進度也會消失。
